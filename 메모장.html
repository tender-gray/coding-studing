HTML

HTML is a system of codes for producing documents for the Internet. HTML is an abbreviation for `hypertext markup language'.


br tag는 enter과 같은 효과를 나타낸다. 또한 br tag는 따로 닫지 않는다.

p tag는 br tag와 다르게 열고 닫아야 한다. p tag의 열림과 닫힘 사이는 하나의 문단으로서 인정을 하게 된다.

여기에 있는 style부터 들어가는 내용은 css의 내용이다. margin은 태그와 태그 사이의 여백을 뜻한다. 뒤에 나오는 -top 은 태그 웨 공간이라는 뜻이다. 80px은 80px만큼 띄우겠다 라는 컴퓨터상의 수치 이다.
